Описание проекта

Сайт педставляет собой веб-приложение для базы данных.

При разработке приложения были использованы технологии: PHP, HTML, JavaScript и Ajax (JQuery).
Возможности приложения:

  1. Регистрация покупателей (для новых пользователей)


  2. Вывод всех игр на главный экран (для всех пользователей)


  3. Вывод персональных данных (для покупателей и администраторов)


  4. Осуществление покупок (для покупателей)


  5. Подтверждение и отправка заказов (для администратора)


  6. Вывод на экран истории заказов (для покупателей)


  7. Вывод на экран статистики (для администратора)


  8. Добавление товаров в корзину (для покупателей)
